# Examples

These aren't installed anywhere automatically -- copy whichever pieces you
want into your own `~/.config/calm-shell/` (see `docs/CONFIGURATION.md` for
the full format guide and file layout).

- `aliases.calm` -- a fuller set of aliases than the bundled defaults
- `functions.calm` -- a couple of small, genuinely useful shell functions
- `themes/calm-mint.calm` -- a second bundled-quality theme, for
  `calm theme set calm-mint` after copying it into
  `~/.config/calm-shell/themes/`
